# Doob Agency Landing Page

A modern and responsive digital agency landing page built using HTML, CSS, and JavaScript. This project focuses on clean design, smooth user experience, and well-structured layout for agency-style websites.

## 🚀 Overview

Doob is a creative agency landing page designed to showcase services, portfolio, and company information in a visually appealing and professional way.

The layout follows common agency website patterns such as:
- Hero section with call-to-action
- Services presentation
- About section
- Portfolio / Gallery
- Blog / News section
- Contact & Footer

## 🎯 Project Goals

- Build a professional agency landing page UI
- Practice responsive web design techniques
- Improve layout structuring using CSS Grid & Flexbox
- Apply clean and scalable CSS architecture
- Enhance visual hierarchy and user experience

## 🛠️ Technologies Used

- HTML5
- CSS3 (Flexbox & Grid)
- JavaScript (Vanilla JS)

## ✨ Features

- Fully responsive design (Mobile, Tablet, Desktop)
- Clean and modern UI design
- Smooth navigation between sections
- Reusable layout structure
- Organized and readable code
- Interactive elements and hover effects
